instanceof مەشغۇلاتچىسى

بۇ مەشغۇلاتچىمۇ typeof قا ئازراق ئوخشىشىپ قالىدۇ،بۇمەشغۇلاتچى قۇرۇلمىلىق فۇنكىسيەنى سىلىشتۈرۈش ئۈچۈن قوللىنىلىدۇ،بۇ مەشغۇلاتچى پەقەت ئۆزى قۇرغان ئوبيكىتنى سىلىشتۇرۇشتا قوللانسا بولىدۇ
ئۆزى ياسىغان ئوبيكىتنى سىلىشتۇرۇش 

function Foo() {}
function Bar() {}
Bar.prototype = new Foo();

new Bar() instanceof Bar; // true
new Bar() instanceof Foo; // true

// This just sets Bar.prototype to the function object Foo,
// but not to an actual instance of Foo
Bar.prototype = Foo;
new Bar() instanceof Foo; // false

instanceof نى تەبئى تىپ بىلەن قوللىنىش 

new String('foo') instanceof String; // true
new String('foo') instanceof Object; // true

'foo' instanceof String; // false
'foo' instanceof Object; // false
